Media queries

Las media queries que, traducido al español, significa "consulta de medios" son breakpoints (puntos de quiebre) utilizados cuando se quiere modificar la disposición del contenido web en función del tipo de dispositivo.

Por ejemplo: Si visualizas está página web desde un teléfono celular, la paleta de colores y la distribución del contenido será diferente a lo que verías si realizaras la consulta desde una computadora.

Esta y otras modificaciones específicas se logran gracias a las media queries.

Turn JS

Es una biblioteca JavaScript que transforma el contenido de la página, haciendo que esta se vea como un libro o revista real utilizando todas las ventajas de HTML5.

Animate Style CSS

Animate.css es una biblioteca de animaciones que permite darle vida o movimiento a los elementos dentro de la página web. Estas amimaciones son super prácticas para hacer énfasis, es decir, que, al detectar el movimiento, el usuario dirija su atención al elemento que se está resaltando.

En esta página web, las imágenes y algunos títulos saltan o aparecen de los lados para demostrar el funcionamiento de la librería.

Wow JS

Wow.js es otra librería de JavaScript que se complementa con animate.css para que las animaciones vayan apareciendo a medida que el usuario hace scroll a la pantalla, es decir, se desliza hacia abajo.

En resumen, lo que hace wow.js es detectar una acción o evento y, cuando la detecta, aplica las animaciones CSS.
